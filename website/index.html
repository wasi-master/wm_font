About WM Font

WM Font: It is a fully authentic font designed from the ground up by creator Arian Mollik Wasi. The font was initially created for the Typeface challenge at Hack Club. Wasi used the online tool Calligraphr to convert his handwriting into a digital font. To write the font, Wasi used a Samsung Galaxy S25 Ultra with a stylus, which allowed him to create precise and detailed strokes. The font contains 207 characters, including uppercase and lowercase letters, numbers, punctuation marks, and special symbols. Wasi personally uses special unicode characters a lot and thus he has included these characters in the font as well. Wasi has a personal website available at wasi-master.github.io where he showcases his coding projects.
License: The font is licensed under the SIL Open Font License (OFL). The code for this website is licensed under the MIT License. Both are open-source licenses that allow you to use, modify, and distribute the font and website code freely, as long as you comply with the terms of the license. The source code can be found on GitHub.

Programmatic Use

You can use WM Font in your web projects by including it via CSS. Here's an example of how to include the font in your CSS: 

      
@font-face {
  font-family: 'WM Font';
  src: url('./WmFont-Regular.woff2') format('woff2'),
      url('./WmFont-Regular.woff') format('woff'),
      url('./WmFont-Regular.ttf') format('truetype'),
      url('./WmFont-Regular.otf') format('opentype'),
      url('./WmFont-Regular.svg#WMFont') format('svg');
  font-weight: normal;
  font-style: normal;
}

After including the font, you can use it in your CSS like this:

      
body {
  font-family: 'WM Font', sans-serif;
}

Make sure to adjust the file paths in the `src` property to point to the correct location of the font files in your project relative to your CSS file.

What is WM Font?

WM Font is a hand drawn font made by Arian Mollik Wasi. It was created for a font making contest called Typeface organized by Hack CLub

Can I use WM Font in my project?

Since WM font is licensed under the SIL Open Font License, you can use it, modify it, and distribute it for personal and commercial projects as long as you follow the terms of the license.

How do I install the font to my computer?

There are download links below, select the ttf or otf format and you should be able to install them to your computer. If you are on Mac, you should double-click the font file then click "Install Font". If you are on Windows, you can right-click the font file and select "Install".
